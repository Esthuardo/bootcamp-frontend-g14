# Ejercicio parta hablar del IMC
### Requerimientos

* [] Definir un formulario con dos cajas de texto. Una para la altura (cm) y otra para el peso (kg)
* [] Mostrar en pantalla el IMC

````````
IMC=peso/altura^2
````````
**Condiciones de IMC**
* Baja: < 18.5
* Normal: 18.5 - 24.9
* Sobre peso: 25 - 29.9
* Obseso > 30
